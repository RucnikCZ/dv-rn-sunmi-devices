# dv-rn-sunmi-devices

Module for controlling part of system functions of Sunmi T2 mini and P2 pro

## Installation

```sh
npm install dv-rn-sunmi-devices
```

## Usage

```js
import { multiply } from "dv-rn-sunmi-devices";

// ...

const result = await multiply(3, 7);
```
